# 課題１ コマンド作成
時間に応じた挨拶を返してくれるプログラム

## 必要なソフトウェア
* python(テスト済みバージョン：3.7~3.11)

## 実行方法
~~~  
git clone git@github.com:Satoshi0412/robosys2024.git
~~~
を入力

~~~
cd tmp/robosys2024
~~~
でリポジトリを移動する

~~~
./kadai1
~~~
で実行
